# Демонстрация AI-риелтора

Рабочая демонстрация голосового AI-риелтора для браузера. Это не телефония и не реальный обзвон: каждый запуск имитирует один звонок одному клиенту из холодной базы.

## Что делает демо

- открывает голосовой разговор через OpenAI Realtime API;
- даёт простой вход/регистрацию по e-mail и паролю;
- AI начинает разговор первым и знает только имя клиента;
- пользователь отвечает голосом свободно;
- AI квалифицирует клиента по шкале: горячий, тёплый, тёпленький, полухолодный, холодный;
- после завершения показывает карточку результата;
- карточки сохраняются в серверной базе отдельно для каждого пользователя.
- сохраняет аудиозапись звонка, чтобы её можно было прослушать в интерфейсе.

## Запуск

Создайте файл `.env`:

```bash
OPENAI_API_KEY=sk-...
PORT=3000
```

Запустите сервер:

```bash
npm start
```

Откройте:

```text
http://localhost:3000
```

Для работы нужен современный браузер с доступом к микрофону.

## Техническая схема

Браузер получает временный Realtime client secret с локального сервера, подключается к OpenAI Realtime API по WebRTC и передаёт микрофон напрямую в живую голосовую сессию. Отдельные STT/TTS сервисы и записанные аудиозаготовки не используются.

Пользователи, сессии, лиды и метаданные записей хранятся в `data/db.json`. Аудиофайлы лежат в `data/recordings`. Это демо-хранилище для быстрого показа, не production-БД.
